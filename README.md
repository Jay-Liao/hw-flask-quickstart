# hw-flask-quickstart
A homework for flask quick start.

## Setup on macOS
```sh
git clone https://github.com/Jay-Liao/hw-flask-quickstart.git
virtualenv -p python3 venv
source venv/bin/activate
pip install -r requirements.txt
```

## Run service
```sh
python run.py
open http://localhost:5000/3/4
```
